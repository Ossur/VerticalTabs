Vertical Tabs Release Notes - 0.0.4

Key bindings

Key bindings for Vertical Tabsare now visible in Preferences (Global->Keys), you can customize bindings for most of the plugin commands. All of the commands have the category setup to Vertical Tabs

Focus on Vertical Tabs filter

Additionally, to the previous binidngs, focus on the Vertical Tabs filter was added - default biding Ctrl+Alt+V. It will allow you to use only keayboard if that is your style. If the view is closed at the moment of using the shortcut, it will be opened at first.
This is idea raised in the GH

Plugin preferences shortcut

The prefereces shortcut is added to view menu, for quicker access.

When used, filtered preferences dialog will be open.
